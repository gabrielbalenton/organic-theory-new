The Single Biggest Opportunity

A URL as clean as the craftsmanship.

Your reviews page is indexed at a URL ending in a long random string instead of something readable like /reviews — a leftover from how the page got created, not something anyone chose. Renaming the page and redirecting the old address is a same-week fix that makes every search result under handcarvedsurfboards.com look as considered as the boards themselves.

What I'd Fix In The First Week

  1. 01Rename the rave-reviews page to a clean, readable URL like /reviews.
  2. 02301-redirect the old hash URL to the new clean one so no existing links break.
  3. 03Resubmit an updated sitemap in Search Console once the redirect is live.
  4. 04Add LocalBusiness schema for the Cocoa Beach shaping shop.
